## 🚀 Quick Start (5 minutes)

```bash
# 1. Navigate to backend
cd backend

# 2. Install dependencies
npm install

# 3. Create .env file (see docs/setup/PACKAGE_JSON_AND_ENV.md)

# 4. Start server
npm run dev

# 5. In another terminal, start frontend
cd frontend
npm install
npm run dev

# 6. Open http://localhost:5173
```
